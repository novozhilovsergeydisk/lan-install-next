# Отчет об изменениях перед коммитом

## Измененные файлы и логика
- **`app/portfolio/[id]/page.tsx`**: 
  - Реализован кастомный Lightbox для полноэкранного просмотра фотографий проекта.
  - Добавлена логика переключения изображений (вперед/назад), закрытия окна и блокировки прокрутки страницы при открытой галерее.
  - Внедрена поддержка клавиатуры (Esc для закрытия, стрелки для навигации).
  - Улучшен UI: добавлен полупрозрачный фон с размытием, счетчик фото и интерактивные кнопки с иконками `lucide-react`.
