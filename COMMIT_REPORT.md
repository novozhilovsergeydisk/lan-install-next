# Отчет об изменениях перед коммитом

## Измененные файлы и логика
- **Рефакторинг платформы**:
  - Совершен полный переход с **Vite (SPA)** на **Next.js 15+ (App Router)**.
  - Удалены устаревшие файлы: `vite.config.ts`, `index.html`, `App.tsx`, `index.tsx`, а также папка `pages/`.
  - Внедрена современная структура папок `app/` для маршрутизации.
- **Маршрутизация и SEO**:
  - `HashRouter` (ссылки с `#`) заменен на полноценную серверную маршрутизацию.
  - Все ссылки обновлены для использования `next/link`.
  - Улучшена индексация сайта поисковыми роботами за счет серверного пререндеринга.
- **Стили и верстка**:
  - Настроено использование **Tailwind CSS v4**.
  - Исправлены проблемы с кастомными цветами бренда (`lanBlue`, `lanGreen`, `lanOrange`).
  - Глобальные стили, шрифты (Roboto) и анимации перенесены в `app/globals.css`.
- **Очистка**:
  - Файлы `.next` и `out` добавлены в `.gitignore`.
  - Удалены все неиспользуемые зависимости из `package.json`.

## Регрессии
Не обнаружено. Команда `npm run build` подтвердила успешную генерацию всех статических страниц.

## Безопасность
Утечек данных нет. Конфигурация проекта приведена к современным стандартам безопасности.
