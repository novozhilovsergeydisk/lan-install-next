# Отчет об изменениях перед коммитом и деплоем

## Измененные файлы и логика
- **`app/calculator/page.tsx`**: 
  - Полный редизайн интерфейса калькулятора.
  - Внедрены «всплывающие» карточки услуг с мягкими тенями и подсветкой при наведении.
  - Обновлены кнопки управления количеством: добавлены плавные переходы, цветовая индикация (зеленый/оранжевый) и анимация появления итоговой цены позиции.
  - Панель итогов стилизована под «стеклянный» дизайн (Glassmorphism) с декоративными элементами и размытием.
  - Улучшена визуализация графиков и индикаторов активности категорий.
- **`components/Header.tsx`**: Изменен порядок пунктов меню — «Портфолио» теперь предшествует «Калькулятору».
- **`app/globals.css`**: Добавлены ключевые кадры и утилита для анимации `shake` (используется при ошибках в формах).
- **`package.json`**: В скрипт `dev` добавлен флаг `--webpack` для обеспечения стабильной работы сервера на macOS и предотвращения сбоев Turbopack.
- **`DEPLOY_PLAN.md`**: Создан план первичной настройки и деплоя на сервер lan-install без использования PM2 (systemd).

## Регрессии
Не обнаружено. Стабильность локального сервера повышена за счет перехода на Webpack в режиме разработки. Сборка `npm run build` подтвердила корректность всех изменений и отсутствие ошибок типизации TypeScript.

## Безопасность
Утечек данных (паролей, ключей, токенов) в публичные файлы нет. Все изменения носят исключительно визуальный, конфигурационный и инфраструктурный характер.
