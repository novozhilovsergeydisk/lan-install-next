# Отчет об изменениях перед коммитом

## Измененные файлы и логика
- **`app/contact/page.tsx`**: Добавлено скрытое поле Honey Pot (`website`) для защиты от автоматического спама. Поле невидимо для реальных пользователей и исключено из навигации клавишей Tab.
- **`app/api/send-message/route.ts`**: Внедрена проверка Honey Pot на стороне сервера. Если скрытое поле заполнено (что характерно для ботов), отправка в Telegram блокируется.
- **Логирование**: Реализована запись данных о заблокированных спам-заявках в локальный файл `spam_logs.txt` в корне проекта для последующего анализа.
- **`.gitignore`**: Файл `spam_logs.txt` добавлен в исключения, чтобы предотвратить утечку логов в публичный репозиторий.

## Регрессии
Не обнаружено. Основная логика отправки для реальных пользователей осталась без изменений.

## Безопасность
Повышена защита от DDoS-атак на бота Telegram и автоматического спама через формы обратной связи. Конфиденциальные логи защищены от попадания в Git.
